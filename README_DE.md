PORT MASTER ist ein fortschrittliches, mehrsprachiges und benutzerfreundliches Tool für Cybersicherheit und Netzwerkmanagement für Linux-Systeme. Verwalten Sie Ihre Firewall (UFW), überwachen Sie aktive Ports und sorgen Sie für Netzwerksicherheit ohne komplexe Terminalbefehle.

Made by RENTER 🚀
✨ Funktionen

    🌍 Mehrsprachig: Englische, türkische, deutsche und russische Oberfläche.

    ⚡ Auto-Installer: Integriert sich mit einer Bestätigung in das System.

    🔒 Firewall-Verwaltung: Ports über UFW mit einem Klick öffnen (Allow) und schließen (Deny).

    🗑️ Regeln löschen: Dauerhaftes Löschen von Regeln aus der UFW-Liste.

    📡 Live-Port-Überwachung (SS): Anzeige aktiver TCP/UDP-Ports in Echtzeit.

    🕵️ Prozesserfassung (LSOF): Finden Sie heraus, welche Anwendungen offene Ports belegen.

    💀 Prozess beenden (Kill): Erzwingen Sie das Beenden von blockierten Prozessen über den Port.

    👁️ Externer Scan (NMAP): Scannen Sie Ihr System aus der Sicht eines Angreifers, um offene Ports zu erkennen.

    🛡️ Sicherheitsprotokolle (UFW Events): Lesen Sie blockierte Verbindungen und IP-Adressen.
