PORT MASTER — это передовой, многоязычный и удобный инструмент кибербезопасности и управления сетью для систем Linux. Управляйте брандмауэром (UFW), отслеживайте активные порты и обеспечивайте безопасность сети без сложных команд терминала.

Made by RENTER 🚀
✨ Особенности

    🌍 Многоязычная поддержка: Английский, турецкий, немецкий и русский интерфейсы.

    ⚡ Автоматическая установка: Интегрируется в систему одним подтверждением.

    🔒 Управление брандмауэром: Открытие (Allow) и закрытие (Deny) портов через UFW в один клик.

    🗑️ Удаление правил: Навсегда удалите любое правило из списка UFW.

    📡 Мониторинг портов (SS): Список активных TCP/UDP портов в реальном времени.

    🕵️ Обнаружение процессов (LSOF): Узнайте, какие приложения занимают открытые порты.

    💀 Завершение процессов (Kill): Принудительное завершение зависших процессов через порт.

    👁️ Внешнее сканирование (NMAP): Сканирование вашей системы с точки зрения злоумышленника.

    🛡️ Журналы безопасности (UFW Events): Чтение заблокированных соединений и IP-адресов.
