PORT MASTER is an advanced, multi-language, and user-friendly cybersecurity and network management tool developed for Linux systems. You can manage your firewall (UFW), monitor active ports, and ensure your network security without dealing with complex terminal commands.

Made by RENTER 🚀
✨ Features

    🌍 Multi-Language Support: English, Turkish, German, and Russian interfaces.

    ⚡ Auto-Installer: Integrates into the system core with a single confirmation.

    🔒 Firewall Management: Open (Allow) and close (Deny) ports via UFW with one click.

    🗑️ Rule Deletion: Permanently delete any rule from the UFW list.

    📡 Live Port Monitoring (SS): List active listening TCP/UDP ports in real-time.

    🕵️ Process Detection (LSOF): Find which applications are occupying open ports.

    💀 Kill Process: Forcefully terminate stuck or unwanted processes via their port.

    👁️ External Scanning (NMAP): Scan your system from an attacker's perspective to detect open ports.

    🛡️ Security Logs (UFW Events): Read blocked connections and IP addresses caught by the firewall.
